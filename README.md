# Marketing Funnel & Conversion Performance Analysis
## Future Interns - Data Science & Analytics Internship
### Task 3 | FUTURE_DS_03

## Objective
Analyze marketing funnel data to identify conversion 
drop-offs and opportunities to improve lead-to-customer 
conversion.

## Tools Used
- Power BI Desktop
- Custom Marketing Funnel Dataset
- GitHub

## Dataset
Custom funnel dataset with 5 stages:
- Visitors: 10,000
- Leads: 4,200
- Signups: 2,100
- Free Trial: 1,200
- Purchases: 650

## Key Insights
- Only 42% of visitors become leads
- 50% of leads complete signup
- 54% of signups start free trial
- Overall conversion rate is only 6.5%
- Biggest drop-off happens at Lead stage

## Key Metrics
- Total Visitors: 10,000
- Total Leads: 4,200
- Total Purchases: 650
- Conversion Rate: 6.5%
- Drop Off Rate: 93.5%
